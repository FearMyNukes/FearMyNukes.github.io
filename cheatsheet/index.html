Forms

Tag Name Description
<form> Form Used to collect and send information to an external source. When submitted information is passed to the source named in the action atribute.
<input> Input Used to collect data can be of type: Number, Range, Text, Radio, Password, and Checkbox
<label> Label The HTML <label> element provides identification for a specific <input> based on matching values of the <input>'s id attribute and the <label>'s for attribute. By default, clicking on the <label> will focus the field of the related <input>.
<select> Select Used to create a dropdown list of choices with multiple option elements

Semantics

Tag Name Description
<header> Header Describes the content at the top of the page <body>.
<nav> Navigation Encapsulates the page's navigational links may contain the website's logo.
<main> Main encapsulates the main content of a page between the header/navigation and the footer areas.
<footer> footer includes the page's footer content at the bottom of the <body>
